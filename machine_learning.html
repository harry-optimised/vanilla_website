Object Detection with Deep Learning

This is an introductory tutorial series to doing object detection with Deep Learning. I cover each of the seminal papers in object detection from 2014 when they first appeared to the state of the art architectures as of 2020. When I first started learning this field, it took me a while to bridge the gap from basic convolutional classifiers to object detectors, in this tutorial series I hope to accelerate that process for you by explaining more clearly the parts that I struggled with the first time around.

The best way to tackle this series is to read it alongside the papers. Except for the first few introductory posts, each post tackles one paper, and I attempt to explain the important bits in detail, so that they can be understood by someone with slightly less background in object detection than the papers assume.

I'm still writing this series right now (January 2021), more posts will be added as they get written.
